## Purpose

Generate comprehensive weekly briefing documents for CEO review, summarizing business operations, key metrics, and action items.

---

---

## Procedure

### Step 1: Data Collection
- [ ] Review `/done` folder for completed tasks
- [ ] Check `/in_progress` for ongoing work
- [ ] Scan `/clients` for client activity
- [ ] Review `/logs` for significant events

### Step 2: Compile Metrics
- [ ] Count tasks completed this week
- [ ] Identify blockers encountered
- [ ] Note client deliverables
- [ ] Calculate skill utilization

### Step 3: Generate Briefing
- [ ] Create briefing using template below
- [ ] Save to `/reports/Weekly_CEO_Briefing_YYYY-MM-DD.md`
- [ ] Link from Dashboard

### Step 4: Notification
- [ ] Update Dashboard with briefing status
- [ ] Log in Skill_Usage_Log
